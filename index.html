Welcome to the GitHub Page of snmp2xml4j

snmp2xml4j is a software library able to transform snmpwalk requests in xml The tool support mib parsing or working with standard OIDs

Authors and Contributors

In 2010, Nikolay Milovanov (@niau) and Vasil Yordanov (@vasil-yordanov) founded netTransformer. A couple of years later we have outsourced snmp2xml4j from the main project and have provided many developers with the opportunity to work with xml in a structured and convenient way.
